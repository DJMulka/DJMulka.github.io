Machine Learning Powered Kitting System

Duration: 6 months
Role: Lead Engineer
Industry: Aerospace
Status: Deployed

Project Demo & Overview

This system was developed for an aerospace customer facing frequent errors in prepacked hardware kits, ranging from incorrect quantities to missing or misplaced components. To address these challenges, I created a connected, augmented reality-driven solution that ensures accuracy and traceability throughout the kitting process.

The system provides real-time AR guidance, visually directing operators to the correct components using a dynamic display interface. To validate each pick, an integrated computer vision system captures an image of the presented parts and uses a machine learning model to automatically identify and verify them. This advanced sensing layer adds an additional safeguard against human error without slowing down the workflow.

By combining AR interaction, intelligent perception, and MES integration, the solution delivers a high level of precision and accountability, dramatically reducing rework, improving kit integrity, and supporting quality-critical aerospace operations.

Technical Implementation

Augmented Reality Machine Learning Computer Vision Cognex Camera Integration REST API VB.NET

Key Features

Machine-Learning Part Identification: A high-resolution camera captures images of selected parts, which are then analyzed using a custom-trained machine learning model to verify correctness in real time.

Augmented Reality Pick Guidance: Operators are visually directed through the kitting process using an AR interface that highlights the correct components, reducing reliance on paper instructions and minimizing confusion.

Connected System Integration: Seamlessly integrates with the customer's plant management system to pull live kit data and push validation results, ensuring full traceability and process alignment. Part and kit labels are automatically generated and printed to automate the kitting process

Results & Impact

The deployment of this system delivered significant improvements including:

  • 32% reduction in time to assemble kits
  • 0 defects achieved across thousands of delivered kits using no fault forward approach
  • Ability to have even untrained operators pack kits

This system has been running successfully on-site since April of 2023.

Challenges & Solutions

Challenge: Kits being delivered with the wrong parts and quantities.

Solution: Use a trained machine learning model to properly identify and count the correct number of parts have been picked.

Challenge: Incorrect parts were ending up in bins labeled for other parts.

Solution: Operators would scan a barcode for the part they wanted to restock and the system would use augmented reality guidance to highlight the correct bin.

Challenge: Manufacturing site cannot run a cloud based model.

Solution: Train a model on the cloud and deploy it locally in the manufacturing site.
