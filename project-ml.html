ML-Powered Kitting System

Duration: 6 months
Role: Lead Engineer
Industry: Aerospace
Status: Deployed

Project Demo & Overview

This is a system that I developed for an aerospace customer that needed help reducing errors in prepacked kits of assembly hardware. Kits were arriving at the factory with wrong quantities, wrong parts or missing parts all together. This system addresses their needs by providing augmented reality based guidance to indicate which parts to pick. This system also validates the correct picks were made by using a computer vision camera to take a snapshot of the presented parts and identify them with a machine learning model.

Technical Implementation

Augmented Reality Machine Learning Computer Vision Cognex Camera Integration REST API VB.NET

Key Features

Machine-Learning Part Identification: On prem deployed Machine-Learning model validated that the correct part and quantity was selected by the operator

Augmented Reality Kitting Guidance: Projected light guides the user to select parts from the appropriate bins ensuring the correct part is chosen

Label Printer & Barcode Scanner Integration: Part bag labels and kit labels are automatically generated and printed by using data from the plant management system to automate the kitting process

Results & Impact

The deployment of this system delivered significant improvements including:

  • 32% reduction in time to assemble kits
  • 0 defects achieved across thousands of delivered kits using no fault forward approach
  • Ability to have even untrained operators pack kits

This system has been running successfully on-site since April of 2023.

Challenges & Solutions

Challenge: Kits being delivered with the wrong parts and quantities.

Solution: Use a trained machine learning model to properly identify and count the correct number of parts have been picked.

Challenge: Incorrect parts were ending up in bins labeled for other parts.

Solution: Operators would scan a barcode for the part they wanted to restock and the system would use augmented reality guidance to highlight the correct bin.

Challenge: Manufacturing site cannot run a cloud based model.

Solution: Train a model on the cloud and deploy it locally in the manufacturing site.
