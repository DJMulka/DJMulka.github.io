Project Overview

I developed this spotlight based warehouse guidance system to assist operators when building assembly kits. The custom application reads a list of picks from the plant's inventory management system (MES) and sends the waypoints to an overhead spotlight that highlights the parts. After picking, the operator uses a barcode scanning glove to verify that they picked the correct part. If the pick was correct, the light flashes green and moves to the next part in the list, if it is not correct it will turn red.
