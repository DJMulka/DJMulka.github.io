Project Overview

In this demo, I showcase an augmented reality-enhanced warehouse guidance system designed to streamline the part-picking process common in manufacturing and logistics environments.

The spotlight system that integrates with the plant's Manufacturing Execution System (MES) to dynamically guide operators through kit assembly tasks. The application reads real-time pick lists and directs an overhead AR spotlight to illuminate the correct part location. To ensure accuracy, operators use a wearable barcode scanner glove to confirm each pick. The system provides immediate visual feedback, flashing green for correct picks and red for errors, enabling fast, intuitive guidance and error prevention.

This solution combines AR, advanced sensing, and industrial connectivity to boost accuracy, reduce training time, and improve overall efficiency on the shop floor.

Key Features

Augmented Reality Spotlight Navigation: Leverages an overhead augmented reality spotlight system, synced with the MES, to visually direct operators to the correct parts in real time, minimizing search time and cognitive load.

Seamless Wearable Validation: Utilizes a barcode-scanning glove for immediate, hands-free part verification, reducing errors and increasing pick efficiency.

Scalable, Connected Infrastructure: The modular design enables easy integration with existing MES platforms and IoT devices, positioning it as a scalable solution for Industry 4.0 environments.

Results & Impact

The implementation delivered measurable improvements across multiple key performance indicators:

  • Reduced Picking Errors by Over 90%
  • Cut Operator Training Time in Half
  • Improved Throughput and Line Readiness
  • Decreased Cognative Load Of Users
